## Partie MongoDB

L'utilisation de MongoDB est présente pour sauvegarder les données de l'application.
Nous utilisons le **Cloud de MongoDB** à l'adresse https://cloud.mongodb.com.

Exemple de `.env` :

```dotenv
MONGODB_USERNAME=USER
MONGODB_PASSWORD=PASS
KEY=key
FRONT_DOMAIN=*
```
